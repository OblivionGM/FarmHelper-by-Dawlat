# FarmHelper by Dawlat

**Версия**: 1.0.0  
**Автор**: Dawlat  
**Платформа**: [MoonLoader](https://blast.hk/moonloader/)

FarmHelper — удобный скрипт для отслеживания и ведения статистики сбора на ферме льна и хлопка в игре [Arizona RP](https://arizona-rp.com/).
Скрипт обеспечивает детальную статистику по собранным ресурсам и возможность логирования.

## Особенности

- Автоматическое отслеживание собранных ресурсов на ферме льна и хлопка
- Подсчёт общей стоимости собранных ресурсов по настраиваемым ценам
- Предупреждение о голоде при сытости ниже 30
- Статистика собраных ресурсов и заработка за выбранный период
- Среднестатистические данные про собраные ресурсы
- Удобные настраиваемые бинды на рутинные задачи
- Автоматизация использования комплексных обедов по одному нажатию
- Отслеживание использования сиропов с таймером
- Поддержка логов в формате JSON

## Установка

1. Скачайте файл `FarmHelper_by_Dawlat.lua` из [репозитория](https://github.com/OblivionGM/FarmHelper-by-Dawlat).
2. Переместите файл `FarmHelper_by_Dawlat.lua` в папку `moonloader` вашей SAMP-сборки.
3. Перезапустите игру если она была запущена.

## Использование

1. После запуска скрипт начнёт отслеживать ваш прогресс автоматически.
2. Для открытия виджета используйте команду `/farm`, для открытия настроек `/fset`.
